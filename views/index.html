Contact Manager API

Welcome to the Contact Manager API, Kindly read the below documentation and make use of it.

GET Request

To get all the Contacts in the Contact Collection, Kindly send a GET request to https://crud-app-yfc0.onrender.com/api/users

GET Single Contact

To get a single Contact in the Contact Collection, Kindly send a GET request along with contact _id https://crud-app-yfc0.onrender.com/api/users/:id

PUT

To update the contact details in the Contact Collection, Send a PUT request along with request body to the URL https://crud-app-yfc0.onrender.com/api/users

POST

To create a new contact, Send a POST request with request body which includes {name,relation,contact_number,avatar_id}, URL : https://crud-app-yfc0.onrender.com/api/users

DELETE

To deleta a contact, Send a DELETE request with the contact _id to the URL : https://crud-app-yfc0.onrender.com/api/users
